# Face Blurring Script (Click-to-blur)
Batch-blurs dynamic human (and human-like) faces in video stimuli of social interaction dyads. Good for small faces and profile views. You click on each face in the first frame of each unique video. The blur is then applied at that fixed position across all frames. Mirror videos (ending in '_m') automatically reuse coordinates from their parent video with horizontally flipped positions. Easy to adjust size and shape of elipses.
